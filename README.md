# TimeSlot Boilerplate

Estructura base para proyecto full-stack con:

- Frontend: React + TypeScript + Vite + MUI
- Backend: Node.js + Express + TypeScript

## Estructura

- `frontend/` → app React
- `backend/` → API Express

## Alias configurados

- Frontend: `@/*` -> `frontend/src/*`
- Backend: `@/*` -> `backend/src/*`

## Scripts

Desde la raíz:

- `npm run dev` → frontend + backend en paralelo
- `npm run dev:frontend`
- `npm run dev:backend`
- `npm run build`

## Inicio rápido

1. `npm install`
2. `cp backend/.env.example backend/.env`
3. `npm run dev`
